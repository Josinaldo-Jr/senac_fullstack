Projeto-01

O objetivo do projeto é construir uma página HTML que apresente o detalhamento dos conteúdos que foi apresentado ao longo da semana. Foi ultilizado recursos vistos em sala para construir a página HTML, esse material também poderá servir de apoio para estudos futuros.

  1. <!DOCTYPE html>
    A declaração de tipo de documento no HTML, também conhecida como DOCTYPE , é a primeira linha de código necessária em qualquer documento, HTML ou XHTML.
  2. <html>
    HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.
  3. <head>
    O elemento HTML "head" providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
  4. Meta tags:
    Meta elementos são tags usadas em documentos HTML e XHTML para fornecer metadados estruturados sobre uma página da Web.
  5. <title>
    O elemento HTML "title" (Elemento HTML Título) define o título do documento, mostrado na barra de título de um navegador ou na aba da página.
  6. <body>
    A tag "body" contém o corpo de um documento HTML, que é exibido pelo navegador em sua janela, ou seja, todo o conteúdo visível do site.
  7. <ol>
    O Elemento HTML "ol" (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados.
  8. <ul>
    O Elemento HTML "ul" (ou Elemento HTML de lista não ordenada) representa uma lista de itens não ordenados, representados por ícones.
  9. <ul>
    O elemento HTML "li" (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista.
  10. <dl> <dt> <dd>
    • O elemento HTML Definition List "dl" engloba uma lista de pares de termos e descrições.
    • O elemento HTML "dt" (ou Elemento HTML de Definição de Termo) identifica um termo na lista de definição.
    • O elemento HTML "dd" fornece detalhes ou uma definição mais completa do termo precedente definido por "dd" numa lista de descrições "dd" .
  11. <b> <i> <strong>
    • O elemento HTML "b" representa um intervalo de texto estilísticamente diferente do texto normal, sem transmitir qualquer importância ou relevância.
    • O elemento HTML "i" representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios.
    • O elemento HTML "strong" dá ao texto uma forte importância, e é tipicamente mostrado em negrito.
  12. <p> <br>
    • O elemento HTML "p" representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco.
    • O elemento HTML quebra-de-linha "br" produz uma quebra de linha em um texto (carriage-return).
